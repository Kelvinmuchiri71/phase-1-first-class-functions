receivesAFunction(callback):

Takes a function as its argument (a "callback").
Calls the provided function using callback().
returnsANamedFunction():

Returns a function with a name (namedFunction).
The returned function can later be called using its reference.
returnsAnAnonymousFunction():

Returns a function without a name (anonymous function).
The returned function can also be stored in a variable and called later.
